# Character Audio Files

Đặt các file âm thanh cho nhân vật Italian brainrot vào thư mục này.

## Danh sách file âm thanh cần thiết:

1. `mamma_mia.mp3` - "Mamma mia!" sound
2. `pizza_time.mp3` - "Pizza time!" sound
3. `its_a_me.mp3` - "It's-a me!" sound
4. `bene.mp3` - "Bene! Molto bene!" sound
5. `ciao.mp3` - "Ciao bella!" sound
6. `fantastico.mp3` - "Fantastico!" sound
7. `perfetto.mp3` - "Perfetto!" sound
8. `game_over_italian.mp3` - "Mamma mia! Game over!" sound

## Ghi chú:

- Các file âm thanh sẽ được phát tự động khi nhân vật xuất hiện
- Nếu file không tồn tại, game vẫn chạy bình thường (không có âm thanh)
- Format khuyến nghị: MP3, độ dài ngắn (1-3 giây)
