# Background Images

Đặt ảnh nền của bạn vào thư mục này.

## Tên file
Đặt tên file ảnh nền là: `wood_background.png` (hoặc `.jpg`)

## Định dạng hỗ trợ
- PNG (khuyến nghị)
- JPG/JPEG

## Kích thước khuyến nghị
- Độ phân giải cao để hiển thị tốt trên mọi thiết bị
- Tỷ lệ khung hình: 9:16 (portrait) hoặc bất kỳ (sẽ được scale để fit)

## Lưu ý
Nếu không có file ảnh, app sẽ sử dụng màu nền mặc định (xanh dương).
